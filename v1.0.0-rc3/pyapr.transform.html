transform

pyapr.transform.maximum_projection(apr, ...)

Compute the maximum intensity projection along an axis.
pyapr.transform.maximum_projection(apr: APR, parts: Union[ByteParticles, ShortParticles, FloatParticles, LongParticles], dim: int, patch: Optional[ReconPatch] = None, method: str = 'auto')

Compute the maximum intensity projection along an axis.

Note: assumes that all particle values are non-negative

Parameters

  • apr (APR) – Input APR data structure

  • parts (FloatParticles or ShortParticles) – Input particle intensities

  • dim (int) – Dimension along which to compute the projection: dim=0: project along Y to produce a ZX plane dim=1: project along X to produce a ZY plane dim=2: project along Z to produce an XY plane

  • patch (ReconPatch, optional) – (optional) If provided, projects only within the image region specified by patch. Otherwise projects through the entire volume. (default: None)

  • method (str) – Specify the projection algorithm (results are identical, but performance may differ). Supported arguments are auto, direct and pyramid.

Returns

out – The computed maximum intensity projection

Return type

numpy.ndarray
